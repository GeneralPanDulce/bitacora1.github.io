JavaScript

Que es JavaScript?

Es un lenguaje de programación que se utiliza para agregar interactividad y dinamismo a las páginas web. Permite crear comportamientos como validaciones de formularios, animaciones, manejo de eventos, entre otras funciones.

DOM

Que es "DOM"?

DOM (Document Object Model) es como un mapa de la página web donde cada elemento (como un botón o una imagen) es una parte del mapa. JavaScript puede usar este mapa para cambiar o mover esas partes de la página.

"Referenciar con JavaScript"

Como se referencia con JavaScript?

Significa usar JavaScript para seleccionar elementos específicos de la página, como un botón o un párrafo, para poder hacer algo con ellos, como cambiar su texto o color. Esto se hace utilizando funciones como getElementById(), querySelector(), o getElementsByClassName()

Propiedad Style

Que es la propiedad "Style"?

Es una propiedad de JavaScript que permite modificar los estilos CSS directamente en un elemento HTML.

Classlist

Que es "Classlist"?

Es una propiedad de los elementos HTML que devuelve una lista de las clases CSS asociadas a un elemento. También permite agregar, quitar o alternar clases dinámicamente.

Eventos

Que son "Eventos"?

Son acciones que pasan en la página, como cuando alguien hace clic o mueve el ratón. JavaScript reacciona a esos eventos para ejecutar código y hacer que pase algo en la página.

If Else

Que es "If Else"?

Es una estructura condicional en JavaScript que permite ejecutar diferentes bloques de código según si una condición es verdadera o falsa

Viewport

Que es "Viewport"?

Es la parte visible de una página web en un dispositivo, que puede variar dependiendo del tamaño de la pantalla. En diseño web, es importante controlar el tamaño del viewport para que las páginas se adapten bien a diferentes dispositivos, como móviles o pantallas grandes.

Medias

Que son los "Medias"?

En CSS, las "media queries" son reglas que permiten aplicar estilos específicos según el tamaño o características del viewport o del dispositivo. Esto se usa para crear diseños web responsivos. Esto ayuda a que las páginas se vean bien tanto en pantallas grandes como en teléfonos pequeños.
